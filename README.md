# Chatbot User Management Dashboard

## Overview
The Chatbot User Management Dashboard is an interactive web application designed to manage users in a chatbot environment. This dashboard provides functionalities for user registration, management, and analytics of user interactions.

## Features
- **User Registration**: Allows new users to sign up with necessary details.
- **User Management**: Admins can view, edit, or delete user profiles.
- **Analytics Dashboard**: Displays user interaction metrics, trends, and insights.
- **Role-based Access Control**: Ensures secure access to different functionalities based on user roles.

## Technologies Used
- **Frontend**: React.js, CSS, HTML
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Authentication**: JWT (JSON Web Tokens)

## Installation
### Prerequisites
- Node.js
- npm (Node Package Manager)
- MongoDB

### Steps to Install
1. Clone the repository:
   ```bash
   git clone https://github.com/ramakrishnaj/chatbot-user-management-dashboard.git
   ```
2. Navigate to the project directory:
   ```bash
   cd chatbot-user-management-dashboard
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```
4. Start the server:
   ```bash
   npm start
   ```
5. Open your browser and go to `http://localhost:3000` to view the dashboard.

## Usage
- After starting the application, you can register a new user by navigating to the registration page.
- Admins can log in and access the user management features from the dashboard.
